Fala galera, blz? Espero que sim!
Vocês conhecem o Windows Azure? Se a resposta for não, dia 29/03 próximo sábado, você terá uma ótima oportunidade para conhecer. Nessa data acontecerá o Global Windows Azure Bootcamp 2014, evento global organizado pelos MVPs da competência Windows Azure de todo o mundo. O evento será realizado em 21 cidades no Brasil, abaixo as cidades mais próximas da região da região de Marília e seus respectivos link's para inscrição. Ah, o evento é totalmente gratuito!
Londrina/PR – http://www.eventick.com.br/gwab-londrina-pr
Maringá/PR – http://www.eventick.com.br/GWAB2014-Maringa-PR
São Paulo (Microsoft Office)/SP – http://www.eventick.com.br/GWAB2014-SaoPaulo-SP
São José dos Campos/SP – http://www.eventick.com.br/GWAB2014-SJCampos-SP

Confira a lista completa das cidades em:
http://lucasromao.azurewebsites.net/pt/global-windows-azure-bootcamp-2014-lista-de-cidades-do-brasil/

Pré Requisitos:

É necessário que o participante leve seu notebook com alguns programas instalados, são eles:

  • Microsoft Visual Studio 2013
  • Windows Azure SDK 2.2 para Visual Studio 2013 .NET
  • Windows Azure PowerShell
  • Windows Azure Training Kit
