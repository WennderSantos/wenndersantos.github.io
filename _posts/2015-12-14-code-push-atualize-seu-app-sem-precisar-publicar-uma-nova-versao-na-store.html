E aê, galera! Tudo bom?

Todo desenvolvedor mobile, seja híbrido ou nativo, sabe como é demorado (apple store) fazer uma publicação de um app. As consequências dessa demora são ruins, claro. Correções de bugs, novas features e etc. uma série de fatores são diretamente afetados.

Seria muito bom ter uma forma de atualização, pelo menos para pequenas correções, fora das stores. Certo? Bom, existe, na verdade existem várias formas, vou apresentar uma delas chamada Code Push.

 

Code Push, atualizações dinâmicas em seu app

O code push é um serviço em nuvem, open source da Microsoft, que permite aos desenvolvedores de aplicações cordova e react native fazerem deploy de updates direto para os usuários.

Funciona como um repositório, onde, você cria sua aplicação e faz atualizações de código. Com pequenas implementações no app, ele ja será capaz de verificar no repositório do code push se existem atualizações. Caso sim, um alerta informará o usuário e perguntará se ele deseja fazer a atualização.

Para quem tiver interesse em ver o passo-a-passo escrito,entre no site do code push que logo na página inicial tem um step-by-step de como fazer toda a configuração.
